---
title: AvatarPair
---

AvatarPairs are nested avatars used to create a parent + child relationship between Avatars.

**Note:** AvatarPair components must be used with `Avatar` components as children.

## Default example

```jsx live
 <AvatarPair my={4}>
  <Avatar src="https://avatars.githubusercontent.com/primer"/>
  <Avatar src="https://avatars.githubusercontent.com/primer"/>
 </AvatarPair>
 ```

## System props

AvatarPair components get `COMMON` system props. Read our [System Props](/system-props) doc page for a full list of available props.

## Component props

AvatarPair components do not receive any additional props besides `COMMON` system props.
